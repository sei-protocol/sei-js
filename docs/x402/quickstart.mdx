## Prerequisites

Before you begin, ensure you have the following:

<AccordionGroup>
    <Accordion title="Node.js & Package Management" icon="square-terminal">
        **Required for running x402 applications:**
        - **Node.js** (v18 or higher) - Required for running x402 packages
        - **npm, yarn, or pnpm** - For package management

        <Check>
            Verify your installation: `node --version` should output v18.0.0 or higher
        </Check>
    </Accordion>

    <Accordion title="Sei Wallet & Test Funds" icon="wallet">
        **Required for testing payments:**
        - **MetaMask** or **Compass Wallet** - For signing transactions
        - **Sei Testnet configured** - Add Sei testnet to your wallet
        - **Test SEI tokens** - Get from [Sei Faucet](https://docs.sei.io/learn/faucet)

        <Info>
            Sei testnet details: Chain ID `1328`, RPC `https://evm-rpc-testnet.sei-apis.com`
        </Info>
    </Accordion>

    <Accordion title="Environment Variables" icon="key">
        **Required configuration for your paid API:**
        - **`FACILITATOR_URL`** - URL of your x402 facilitator service
        - **`ADDRESS`** - Your Sei wallet address (0x format) to receive payments

        <Warning>
            Keep your environment variables secure and never commit them to version control
        </Warning>
    </Accordion>

</AccordionGroup>

## Create Your First Paid API

<Steps>
    <Step title="Install x402 packages">
        Create a new project and install the required packages:

        <CodeGroup>
            ```bash npm
            mkdir my-paid-api && cd my-paid-api
            npm init -y
            npm install x402-express @coinbase/x402 express dotenv
            ```

            ```bash yarn
            mkdir my-paid-api && cd my-paid-api
            yarn init -y
            yarn add x402-express @coinbase/x402 express dotenv
            ```

            ```bash pnpm
            mkdir my-paid-api && cd my-paid-api
            pnpm init -y
            pnpm add x402-express @coinbase/x402 express dotenv
            ```
        </CodeGroup>

        <Check>
            **Expected outcome:** Project directory created with x402 dependencies installed.
        </Check>
    </Step>

    <Step title="Configure ES Modules">
        Update your `package.json` to enable ES modules:

        ```json
        {
          "name": "my-paid-api",
          "version": "1.0.0",
          "type": "module",
          "main": "server.js",
          "scripts": {
            "start": "node server.js"
          },
          "dependencies": {
            "x402-express": "^0.6.0",
            "@coinbase/x402": "^0.6.0",
            "express": "^5.1.0",
            "dotenv": "^17.2.2"
          }
        }
        ```

        <Check>
            **Expected outcome:** Package.json configured for ES modules.
        </Check>
    </Step>

    <Step title="Create environment configuration">
        Create a `.env` file in your project root:

        ```env
        FACILITATOR_URL=http://localhost:3000/facilitator
        ADDRESS=0x1234567890123456789012345678901234567890
        ```

        <Warning>
            Replace `ADDRESS` with your testnet Sei wallet address. The `FACILITATOR_URL` should point to your facilitator
            service, but having a valid one is not required for this tutorial. If you want to
            test with a facilitator, you can run one locally. See the [Facilitator Example](/facilitators/example)
        </Warning>

        <Check>
            **Expected outcome:** Environment variables configured for your API.
        </Check>
    </Step>

    <Step title="Create a simple paid API server">
        Create `server.js` with a protected API endpoint:

        ```javascript
        import {config} from "dotenv";
        import express from "express";
        import {paymentMiddleware} from "x402-express";
        config();

        const facilitatorUrl = process.env.FACILITATOR_URL;
        const payTo = process.env.ADDRESS;

        if (!facilitatorUrl || !payTo) {
          console.error("Missing required environment variables FACILITATOR_URL and ADDRESS");
          process.exit(1);
        }

        const app = express();

        app.use(
          paymentMiddleware(
            payTo,
            {
              "GET /weather": {
                // USDC amount in dollars
                price: "$0.001",
                network: "sei-testnet",
              }
            },
            {
                url: facilitatorUrl,
            },
           ),
        );

        app.get("/weather", (req, res) => {
        res.send({
            report: {
                weather: "sunny",
                temperature: 70,
            },
          });
        });

        app.listen(4021, () => {
          console.log(`Server listening at http://localhost:${4021}`);
        });

        ```

        <Check>
            **Expected outcome:** Express server configured with x402 payment middleware.
        </Check>
    </Step>

    <Step title="Start and test your paid API">
        Start your server:

        <CodeGroup>
            ```bash npm
            npm start
            ```

            ```bash yarn
            yarn start
            ```

            ```bash pnpm
            pnpm start
            ```

            ```bash direct
            node server.js
            ```
        </CodeGroup>

        In another terminal, test the protected endpoint:

        ```bash
        curl http://localhost:4021/weather
        ```

        <Check>
            **Expected outcome:** You should receive a `402 Payment Required` response with payment details, or an error if the facilitator is not running.
        </Check>

        <Info>
            If you get a connection error, make sure your facilitator service is running at the configured URL, or set up a local facilitator using the [Facilitator Example](/x402/facilitators/example).
        </Info>

        Example successful 402 response:
        ```json
			{
				"x402Version": 1,
				"error": "X-PAYMENT header is required",
				"accepts": [
				{
					"scheme": "exact",
					"network": "sei-testnet",
					"maxAmountRequired": "1000",
					"resource": "http://localhost:4021/weather",
					"description": "",
					"mimeType": "",
					"payTo": "0x1234567890123456789012345678901234567890",
					"maxTimeoutSeconds": 60,
					"asset": "0x4fCF1784B31630811181f670Aea7A7bEF803eaED",
					"outputSchema": {
						"input": {
							"type": "http",
							"method": "GET",
							"discoverable": true
						}
					},
					"extra": {
					"name": "USDC",
					"version": "2"
					}
				}
				]
			}
        ```
    </Step>

</Steps>
