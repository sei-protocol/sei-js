## What is SeiJS?

SeiJS is a monorepo that contains multiple NPM libraries for writing applications that interact with Sei. It provides comprehensive tools for both EVM and Cosmos development on the Sei blockchain.

## Setting up

The first step to world-class documentation is setting up your editing environments.

<CodeGroup>

```bash npm
npm install @sei-js/evm @sei-js/cosmjs
```

```bash yarn
yarn add @sei-js/evm @sei-js/cosmjs
```

```bash pnpm
pnpm add @sei-js/evm @sei-js/cosmjs
```

</CodeGroup>
