## Welcome to SeiJS

SeiJS is a comprehensive TypeScript SDK for building applications on Sei Network. It provides everything you need to interact with both Cosmos and EVM ecosystems on Sei, from wallet management to advanced DeFi integrations.

## Package Ecosystem

SeiJS is organized as a monorepo with specialized packages for different use cases:

### Core Packages

<AccordionGroup>
  <Accordion title="@sei-js/cosmos" icon="atom">

    Essential types, encoding utilities, and REST client for Cosmos ecosystem development.
    
    - TypeScript definitions for Sei modules
    - Protobuf encoding/decoding
    - REST API client
    - Transaction utilities
    
    [View Documentation ‚Üí](/cosmos/introduction)
  </Accordion>

  <Accordion title="@sei-js/evm" icon="ethereum">

## Architecture Overview

SeiJS is designed with modularity and flexibility in mind:

```mermaid
graph TB
    A[Your Application] --> B[@sei-js/evm]
    A --> C[@sei-js/cosmjs]
    A --> D[@sei-js/cosmos]
    
    B --> E[Wagmi/Viem/Ethers]
    C --> F[CosmJS]
    D --> G[Sei Modules]
    
    H[@sei-js/create-sei] --> A
    I[@sei-js/ledger] --> B
    I --> C
    J[@sei-js/registry] --> B
    J --> C
    K[@sei-js/sei-global-wallet] --> B
    L[@sei-js/confidential-transfers] --> B
```

### Quick Installation

<CodeGroup>

```bash All Packages
npm install @sei-js/cosmos @sei-js/evm @sei-js/cosmjs
```

```bash EVM Only
npm install @sei-js/evm ethers viem wagmi
```

```bash Cosmos Only
npm install @sei-js/cosmos @sei-js/cosmjs @cosmjs/stargate
```

```bash CLI Tool
npm install -g @sei-js/create-sei
```

</CodeGroup>

### Create Your First Project

<Steps>
  <Step title="Install CLI">
    ```bash
    npm install -g @sei-js/create-sei
    ```
  </Step>
  
  <Step title="Create Project">
    ```bash
    create-sei my-sei-app
    ```
  </Step>
  
  <Step title="Choose Template">
    Select from available templates:
    - **Next.js + EVM**: React app with Ethereum tooling
    - **Next.js + Cosmos**: React app with CosmJS
    - **Vanilla TypeScript**: Pure TypeScript project
  </Step>
  
  <Step title="Start Building">
    ```bash
    cd my-sei-app
    npm run dev
    ```
  </Step>
</Steps>
