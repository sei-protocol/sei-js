## Type Definitions

All packages include comprehensive TypeScript type definitions. Key type categories:

### Transaction Types
- `TxResponse` - Transaction execution results
- `SignDoc` - Transaction signing documents
- `Fee` - Gas fee structures
- `Coin` - Token amount representations

### Client Types
- `SigningStargateClient` - Enhanced CosmJS client
- `SeiCosmosClient` - REST API client
- `ConfidentialTransferClient` - Privacy client

### Utility Types
- `Address` - Address validation and conversion
- `PubKey` - Public key representations
- `Signature` - Cryptographic signatures

## Contributing

The API documentation is automatically generated from TypeScript source code. To contribute:

1. **Fork the repository** on GitHub
2. **Make your changes** with proper TypeScript documentation
3. **Run tests** to ensure compatibility
4. **Submit a pull request** with detailed description

### Documentation Standards

- Use JSDoc comments for all public APIs
- Include usage examples in documentation
- Provide type definitions for all parameters
- Document error conditions and handling
